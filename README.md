# Dark Side of the Moon Oracle

## Description
The "Dark Side of the Moon Oracle" is a unique tarot reading application that combines the traditional art of tarot with modern technology to provide users with insightful and context-aware interpretations of their readings. 

## Features
- **78-card tarot deck**: A complete deck featuring all major and minor arcana cards.
- **6+ spread types**: Different tarot spreads to cater to various types of questions and insights.
- **Context-aware interpretations**: Detailed explanations based on the card placement and user queries.
- **Reading history**: Users can save and view their past readings.
- **Animations**: Engaging animations enhance the user experience during readings.

## How to Use
1. Navigate to the `index.html` file to load the landing page.
2. Select a spread type and enter your question in the provided input.
3. Click 'Start Reading' to see your tarot cards drawn and interpreted based on their positions.
4. Review your reading and save it if desired.

## File Structure
- `index.html`: The landing page of the application.
- `oracle.html`: The main application interface for conducting readings.
- `deck-data.js`: Contains the data for the tarot deck, including card images and meanings.
- `card-meanings.js`: Script that holds the meanings of each tarot card.
- `spreads.js`: Defines the different tarot spreads available for readings.
- `oracle-main.js`: Main logic for the oracle app, handling user interactions and readings.
- `oracle-styles.css`: CSS file for styling the application.

## Setup Instructions
1. Clone the repository: `git clone https://github.com/hardleyzenn-lab/3rdeye.git`
2. Navigate into the project directory: `cd 3rdeye`
3. Open `index.html` in your web browser to start using the oracle.

## Customization Guide
- To change the logo, replace `LOGO_URL_HERE` in the HTML file with the URL of your logo image.
- For customizing card images, update the URLs in `deck-data.js` file to point to your own image assets.
